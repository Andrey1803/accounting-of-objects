# distribution — Готовая программа для распространения

Содержит собранную .exe программу, готовую к запуску без установки Python.

## Как собрать .exe
Из корня проекта выполните:
```bat
pyinstaller build.spec
```
Результат появится в `dist/server/` — скопируйте содержимое сюда.

## Что должно быть в папке
```
distribution/
├── server.exe              # Скомпилированная программа
├── _internal/              # Внутренние файлы (генерируется PyInstaller)
├── templates/              # HTML-шаблоны
├── static/                 # Статика (CSS, JS, иконки)
├── data/                   # Данные (пустая, создаётся при запуске)
├── backups/                # Бэкапы (пустая, создаётся при запуске)
└── ЗАПУСТИТЬ.bat           # Скрипт запуска
```

## Запуск
Дважды кликните **ЗАПУСТИТЬ.bat** или **server.exe**

## Распространение
Эту папку можно:
- Запаковать в ZIP и отправить пользователям
- Скопировать на флешку
- Распаковать на любом компьютере с Windows
- Python НЕ требуется — всё включено в .exe
